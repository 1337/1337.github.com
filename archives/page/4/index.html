Setting Up Mypy

mypy is a static type checker for python.

Installing mypy is simple enough:

pip install mypy

But running it, hmm…

$ mypy .
manage.py:11: error: No library stub file for module 'django.core.management'
foo/wsgi.py:17: error: No library stub file for module 'django.core.wsgi'
foo/urls.py:5: error: Cannot find module named 'djangojs.views'
foo/urls.py:5: error: Cannot find module named 'djangojs'
foo/urls.py:6: error: No library stub file for module 'django.conf'
foo/urls.py:7: error: No library stub file for module 'django.conf.urls'
foo/urls.py:8: error: No library stub file for module 'django.contrib'
foo/urls.py:9: error: No library stub file for module 'django.views.generic.foo'
...
(Goes on for a hundred thousand lines)

So I did what everyone else did and added --ignore-missing-imports to the command line:

$ mypy . --ignore-missing-imports
foo/admin.py:187: error: "Callable[[Any, Any, Any], Any]" has no attribute "short_description"
foo/admin.py:199: error: "Callable[[Any, Any, Any], Any]" has no attribute "short_description"
foo/admin.py:255: error: "Callable[[Any, Any], Any]" has no attribute "short_description"
foo/admin.py:256: error: "Callable[[Any, Any], Any]" has no attribute "admin_order_field"
foo/admin.py:270: error: "Callable[[Any, Any], Any]" has no attribute "short_description"
foo/admin.py:271: error: "Callable[[Any, Any], Any]" has no attribute "admin_order_field"
foo/admin.py:285: error: "Callable[[Any, Any], Any]" has no attribute "short_description"
foo/admin.py:286: error: "Callable[[Any, Any], Any]" has no attribute "allow_tags"

Still thousands of errors!
It turns out every Django app that writes custom admins is affected by this check. I guess I will need to ‘# type: ignore’ them all or try some stub, like mypy-django.

Living in an Aprtment: Pros and Cons

My landlord is letting his place go on sale. This also means my eviction is in order (and, given the reason, has a very generous notice required by law). It is time to revisit what it was like to live in an apartment, and what people with similar experiences gained/missed.

Pros

  1. Stairs optional.
  2. Parking is usually underground. With locks.
  3. Gated entry guaranteed.
  4. Usually more to see out of windows (unless you rent the basement)
  5. Laundry en suite or in building is almost guaranteed.
  6. There may be other amenities like a fitness room, grocery store, and even a pool. Excellent for people wanting/waiting to die here.
  7. The power will probably never go out. If the power goes out, poor old grannies will be trapped, and they probably won’t let that happen.
  8. No snow. No grass.

Cons

  1. You have to share the lift. And if anyone comes in, you have to pretend you know them and smile.
  2. Possible noise from upstairs or downstairs.
  3. Possible noise from the building itself (if central air)
  4. Really troublesome to move in and out.
  5. It takes minutes rather than seconds to leave the building. Add more time is driving your car out of that underground parking.
  6. Mailboxes are small, and no one leaves your packages at your door. Because they can’t get to your door.
  7. Setting off the smoke alarm might cause the entire building to evacuate.
  8. No space for car washes.
  9. Much higher likelihood of someone sniffing your packets and hacking your Wi-Fis.
  10. Way more diseases.
  11. No more sound of raindrops from the roof.
