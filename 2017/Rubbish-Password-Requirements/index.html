Rubbish Password Requirements

  • “The password must contain at least one UTF-32 character.”
  • “Hashing completed too quickly. Please try a different password.”
  • “The password is too common in our database. Please select another one.”
  • “Enter your two-factor authentication token below. (It is 200-characters long, and it expires in 30 seconds.)”
