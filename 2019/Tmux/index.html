Tmux

After press Ctrl+B, followed by

  • ?: show help.
  • c: make a new window (tab).
  • p: go to the previous window (tab).
  • 0 to 9: select that window (tab).
  • %: make a new pane.
  • left or right: select that pane.
  • Alt+1: evenly resize the panes.
